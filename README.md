# RocketFlightSimulation
Celem projektu jest zaimplementowanie mechanik lotu (siła ciągu, zużycie paliwa, fizyka obrotu), prostego UI do sterowania oraz narzędzi do eksperymentów z parametrami pojazdu.

# Zrealizowane funkcjonalności
* Prefab rakiety z parametrami fizycznymi
* Sterowanie ciągiem
* Telemetria na ekranie (prędkość, wysokość)
* Możliwość zoomu kamery / dopasowania widoku (kamera śledząca)
* Edytowalne parametry w Inspectorze — szybkie eksperymenty z balansem lotu
